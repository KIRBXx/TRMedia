Internet is full of boring popups. So, it's a time to break this trend. That's why I created "Layered Popups". With this script you can realize your imagination and make unique multi-layers popup windows. You can create multiple layers with whatever-you-want content and add custom transitions/animations for each layer. You can embed AJAX-ed subscription form which works with autoresponders (MailChimp, GetResponse, iContact, Campaign Monitor, AWeber) and collect users' names/e-mails. You can raise popup window on page load or show it on click event. You can make your website more attractive and pay users attention to certain stuff.

<strong>This is standalone version of the script. Popups can be easily integrated into any website. All you need to do is to add couple lines of code.
For WordPress-driven websites try <a target="_blank" href="http://codecanyon.net/item/layered-popups/5978263">Layered Popups - WordPress Plugin</a>.</strong>

<h3>Features</h3>
<ul>
<li><strong>Unlimited number of popups:</strong> create as many popup as you wish.</li>
<li><strong>Multiple layers:</strong> each popup can contain any number of layers.</li>
<li><strong>Custom layer duration/animation:</strong> customize layer duration/animation with ease.</li>
<li><strong>600+ Google Fonts:</strong> use any of 600+ webfonts.</li>
<li><strong>Subscription form:</strong> AJAX-ed name/e-mail subscription form.</li>
<li><strong>Autoresponders supported:</strong> MailChimp, GetResponse, iContact, Campaign Monitor and AWeber (beta) supported.</li>
<li><strong>Export/import feature:</strong> you can easily export popup settings from one website and import to another website.</li>
<li><strong>On-page-load event handler:</strong> raise any popup on-page-load event.</li>
<li><strong>Auto open delay:</strong> popup window might be opened with delay.</li>
<li><strong>Display modes:</strong> you can display popups every time, once per session or only once.</li>
<li><strong>Remember subscribed visitors:</strong> script set cookie on user machine to avoid repeated popups for subscribers.</li>
<li><strong>On-click-event handler:</strong> show any popup on-click event.</li>
<li><strong>CSV Export:</strong> list of subscribers can be exported as CSV-file to be used with any newsletter systems.</li>
<li><strong>Set of predefined popups:</strong> use any of 11 predefined popups as a template for your work.</li>
<li><strong>Design your own popups:</strong> it's easy - check out <a href="http://www.youtube.com/watch?v=3RFgeTWAPDc">video tutorial</a>. :-)</li>
<li><strong>Responsive design:</strong> popup window looks nice on small screen devices.</li>
<li><strong>Secure admin panel:</strong> create and handle popups through secure admin panel.</li>
<li><strong>Realtime preview:</strong> edit popup/layers and view changes immediately.</li>
<li><strong>Cross domain access supported:</strong> you can install script on one domain and use it with unlimited number of domains.</li>
</ul>

<h3>Installation</h3>
Let's imagine that you have website <code>http://www.website.com/</code> and you want to install script there.
<ol>
<li>Create folder <code>layered-popups</code> (use any other name) in root of your domain. Once created it can be reached by URL: <code>http://www.website.com/layered-popups/</code></li>
<li>Make sure that folder <code>layered-popups</code> has permissions <code>0755</code> and php-files <code>0644</code>.
<li>Edit <code>inc/config.php</code> and set MySQL database parameters.</li>
<li>Go to admin panel <code>http://www.website.com/layered-popups/</code> using default login/password (login: <code>admin</code>, password: <code>admin</code>), configure script and create popups.</li>
<li>Read <code>How To Use</code> section in admin panel.
</ol>
